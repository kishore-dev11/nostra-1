# 🛍️ Nostra - E-commerce Website

## 📌 Project Overview

**Nostra** is a fully responsive e-commerce website built using **HTML, CSS, and JavaScript**.
This project focuses on creating a clean UI and implementing dynamic features like **search** and **filter functionality** using JavaScript.

---

## 🎯 Assignment Requirements

* Build a responsive website using:

  * HTML
  * CSS
  * JavaScript

---

## 📄 Pages Included

* ✅ **Home Page** (`index.html`)
* ✅ **Collections Page** (`collections.html`)
* ✅ **Contact Us Page** (`contact.html`)

---

## ⚙️ Features & Functionality

### 🔍 Search Functionality

* Users can search for products on the **Collections page**
* Results update dynamically based on input

### 🎯 Filter Functionality

* Products can be filtered by category (e.g., Shirts, Pants)
* Implemented using JavaScript DOM manipulation
